-	Data time series, suatu himpunan data engatmatan yang dibangun dalam urutan waktu

-	3 jenis uji stasioneritas  : plotting line graph of the data, plotting Rolling Statistics, Dickey Fuller Test.
-	CrossValidation(CV),  metode statisktik yang dapat digunakan untuk mengevalusasi kinerja model atau algoritma dimana daa  dibagi menjadi data training dan testing,contohnya K-Fold Cross Validation.

Thu, 7 Apr

saya telah mengikuti pembelajaran
 Reinforcement Learning Belajar dengan Penguatan

- Bagian Q Learning dan Deep Q Learning
- Q Learning => termasuk dalam kategori Model-Free RL Algorithm.
- Q Learning => Agent belajar menggunakan evaluation function yang bergantung pada sekumpulan state dan sekumpulann action.
- Pada Q Learning dilakukan 2 iterasi yaitu : Policy iteration dan Value iteration
- Tahap Awal elemen matrix Q-table akan ditentukan bernilai 0 semuanya. Q-table akan di update sejalan dengan bertambahnya jumlah episode.
- Secara prinsip semakin banyak iterasi episode akan semakin baik Q-table nya.
- Q-table ini dapat dianalogikan seperti 'memori' bagi agent. Agent dapat menentukan path optimal berdasarkan nilai yang tertera pada Q-table.
