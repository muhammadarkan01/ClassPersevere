Struktur data dalam python :
1. Tipe data primitive adalah struktur data dasar, blok bangunan untuk manipulasi data dan berisi nilai data yang murni dan sederhana. Terdiri dari Numbers (integer, float, complex), String, dan Boolean. 
2. Tipe data Non primitive adalah berasal dari primitive data digunakan untuk menyimpan kumpulan nilai. Terdiri dari List, Tuple, Dictionary, dan Set. 
Primitive Data Types
- type() fungsi digunakan untuk mengetahui kelas mana yang dimiliki variabel atau nilai. 
- Bilangan Kompleks ditulis dalam bentuk a+bj, a adalah bagian real dan b adalah bagian imajiner. 
- String adalah teks diapit dengan tanda kutip tunggal atau ganda.
- Boolean adalah berupa dua nilai benar atau salah. 
Non Primitive Data Types 
- List berisi item yang dipisahkan dengan koma dan diapit dalan tanda kurung siku. Pengindeksan pada list dimulai dari sisi kiri ke kanan. Namun python juga mendukung pengindeksan negatif (kanan ke kiri). List memiliki berbagai metode yang tersedia yang dapat digunakan untuk memanipulasi list itu sendiri seperti append(), insert(), remove(), extend(),  pop(), dll. 
- Tuple berisi item yang dipisahkan dengan ka dan diapit salam tanda kurung. Tuple tidak mendukung modifikasi sehingga merupakan objek yang tidak dapat diubah.
- Dictionary dideklarasi menggunakan simbol {}. Dictionary juga bisa berubah yaitu kita dapat memodifikasi baik menambahkan, mengurangi, ata mengubah elemen dictionary setelah pembuatan. 
- Tuple tidak dapat diubah sedangkan list, dictionary , dan set dapat diubah. 
   String merupakan primitive data types tapi mendukung banyak fungsi dari koleksi (list dan tuple). String adalah kumpulan abjad, kata atau karakter lainnya. String multi barus dilambangkan menggunakan tanda kutip tiga. String merupakan objek yang inmutable sehingga sekali di deklarasi kita tidak dapat mengubah String tersebut.

Functions di Python : 
- User defined functions
Python memungkinkan pengguna untuk membuat fungsi yang ditentukan pengguna.
- Buit-in functions
Fungsi python yang sudah tersedia sejak melakukan instalasi python tanpa harus membuat fungsi lagi
- Lambda functions
Fungsi lambda juga disebut fungsi anonim atau tanpa nama, biasanya digunakan untuk mendefinisikan logika sederhana
User defined functions in python
- Fungsi adalah metode utama untuk menata kode dan dapat digunakan kembali dalam python
- Blok pernyataan yang menerima beberapa argumen, melakukan beberapa fungsi dan menyediakan output
- Def digunakan untuk membuat fungsi yang ditentukan pengguna Python 
- Hasil dari fungsi dinyatakan dengan pernyataan return
Built-in functions in Python 
Standard
- print()- Mencetak objek ke text stream file, dipisahkan oleh sep dan diikuti oleh akhir
- len()-Mengembalikan panjang (jumlah item) dari suatu objek.
- type()- Dengan satu argumen, kembalikan tipe objek. Id()- Mengembalikan "identitas" suatu objek.
- Input()- Jika argumen prompt hadir, itu ditulis ke output standar tanpa tambahan baris baru. 
- help()- Panggil sistem bantuan bawaan.
Operasi Matematika
- min()- Mengembalikan item terkecil dalam iterable atau terkecil dari dua atau lebih argumen
- max()- Mengembalikan item terbesar dalam Iterable atau terbesar dari dua atau lebih argumen.
- sum()- Menjumlahkan awal dan item dari iterable dari kiri ke kanan dan mengembalikan total. 
- pow() - Kembalikan base ke power exp
- round()- Mengembalikan angka yang dibulatkan ke presisi ndigit setelah titik desimal
- abs()- Mengembalikan nilai absolut dari suatu angka.
Type Conversion
- Primitive type conversion: int(), float(), bool(), str(), complex()
- Non-primitive type conversion: list(), tuple(), dict(), set()
- Metode conversion lainnya: ascii(), bin(), bytes(), hex(), chr(). bytearray(), ord()
Utility based functions
- Sequence based functions: zip(), range(), map(), filter(). reversed(), enumerate(), iter(), sorted(), next()
- OOPS related functions: hasattr(), getattr().setattr(), issubset(), isinstance(), object(). classmethod(), staticmethod(), super(), dir(),
String Manipulation
- Dalam Python, String adalah array byte yang mewakili karakter Unicode. Unicode adalah standar pengkodean karakter Internasional yang mencakup berbagai bahasa, skrip, dan simbol. Setiap huruf, angka atau simbol memiliki unicode yang unik. Unicode adalah perpanjangan dari ASCII yang memungkinkan lebih banyak karakter untuk diwakili.
- String baris tunggal dapat dibuat menggunakan tanda kutip tunggal atau ganda. String multiline dapat dibuat menggunakan tanda kutip tiga.
- Python menyediakan banyak seperangkat operator, fungsi, dan metode untuk bekerja dengan string.
String manipulation menggunakan operators :
- Operator + melakukan operasi matematika dengan Python. Cukup menulis dua literal string bersama-sama dan menggabungkannya.
- Operator * dapat digunakan untuk mengulang string beberapa kali.
- Operator "in" dapat digunakan untuk memeriksa apakah substring ada di dalam string atau tidak
    Python mendukung penggunaan operator seperti +, *, "in" dan "not in untuk memanipulasi string, selain tu string python memiliki banyak metode bawaan yang dapat digunakan untuk memanipulasi string.
String manipulation menggunakan built-in methods
Contoh metode : capitalize(), count(), find(), index(), replace(), split(), strip(), dll.

Generators adalah jenis fungsi khusus yang tidak mengembalikan nilai tunggal, melainkan mengembalikan objek iterator dengan urutan nilai. Dalam fungsi generator, yield statement biasanya digunakan daripada return statement. Python Generator membantu dalam menghemat memori dengan melakukan pemrosesan data secara efisien termasuk dalam melakukan perhitungan
Syntax : 
def generator function(arg1, arg2,…
argN):
<statements>
yield value1
yield value2

Lambda function

lambda arguments: expression

- except: Code yang dieksekusi jika terdapat kesalahan 
- else : Code yang dieksekusi Jika tidak terdapat kesalahan
- finally : Code yang selalu akan dijalankan di akhir
