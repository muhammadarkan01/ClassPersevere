Belajar Python baik itu belajar mandiri melalui W3School dan juga diberikan contoh dari guru secara langsung tentang struktur data yang dibangun dengan Python. Dari W3Schools kita disuruh belajar dari intro sampai kamus. Dari Pengenalan ke variabel tidak ada sesuatu yang baru untuk dipelajari. Kecuali betapa bergunanya memiliki variabel global yang dapat mempengaruhi output keseluruhan program jika kita menginginkannya. Ada beberapa tipe data utama dalam Python seperti string, integer, float, boolean, complex, list, tuple, rage, dict, set, dll. Yang paling menarik adalah belajar tentang struktur data Python, khususnya 'List Comprehensions' yang meminimalkan penggunaan kode misalnya untuk mengulang daftar. Biasanya kita harus mengetikkan perulangan for, tetapi hanya dengan menulis 'x untuk x dalam daftar..(pernyataan)' kita dapat mencapai tujuan yang sama. Daftar itu sendiri mirip dengan array. Tuple seperti daftar yang dipesan dan memungkinkan duplikat tetapi Tuple tidak mengizinkan item ditambahkan atau dihapus. Untuk mengubah item, pertama-tama harus dikonversi ke daftar kemudian diubah kembali menjadi tupel. Set berbeda dari daftar dan tuple yang tidak diurutkan, tidak dapat diubah dan tidak mengizinkan duplikat tetapi masih dapat menambahkan item baru ke set. Kamus digunakan untuk menyimpan nilai data dalam pasangan kunci dan nilai. Ini memungkinkan perubahan pada item, dipesan, dapat berisi tipe data apa pun juga., tetapi tidak mengizinkan duplikat. Selain karakteristik umum dari tipe data ini, ada juga penjelasan tentang metode yang dapat kami berikan pada tipe data tetapi beberapa metode mungkin tidak berfungsi dengan tipe data lain karena sifatnya.
