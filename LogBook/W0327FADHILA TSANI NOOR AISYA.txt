Aljabar Linier : Vektor
- Vektor adalah besaran yang memiliki nilai dan arah. 
- Dalam ilmu data, vektor adalah kumpulan nilai dari variabel tertentu. 
- Skalar adalah konstanta atau besaran yang memiliki nilai. 
- Vektor berdimensi sama dapat dioperasikan 
- Penjumlahan vektor : 
        a+b = (a1+b1,…,an+bn)
- Perkalian saklar : 
         ka = (ka1,…,kan)
- Dot product
         a.b = £ai.bi = a1.b1 + a2. b2+…+an.bn
Jarak dan sudut antar vektor : 
- Jarak Euclidean
         d(a,b) = akar dari £(ai-bi)^2
- Jarak Manhattan
         d(a,b) = £|ai-bi|
- Sudut antar vektor (cosine similarity)
         Cos sudut = vektor a. vektor b / ||vektor a|| ||vektor b||

Aljabar Linier : Nilai dan Vektor Eigen
- Eigen berasal dari bahasa Jerman yang berarti karakteristik. Vektor Eigen adalah vektor yang menjadi karakteristik sebuah matriks dimana arahnya tidak berubah meski dilakukan transformasi
- Sementara nilai eigen adalah bilangan yang berasosiasi dengan panjang vektor yang berubah setelah ditransformasi oleh matriks
- A adalah matriks persegi, x adalah vektor eigen, dan A adalah nilai Eigen 
         Ax = Ax

         Ax - AIx = 0
         (A-AI)x = 0
- Jika x bukan vektor 0, maka kita bisa menemukan nilai eigen dengan menghitung determinant matriks (A - XI). Lalu mensubtitusi nilai eigen tsb ke Ax = Ax untuk menemukan vektor eigennya 
Kalkulus : Fungsi dan Gradient
- Fungsi adalah aturan yang memetakan antara 2 himpunan
- Gradient adalah besaran perubahan atau kemiringan
- Gradient = change in Y / change in X
- m = Y1-Y2 / X1-X2
- Persamaan garis lurus :
         y = mx + c
         c = nilai y saat x =0
Contoh penggunaan Gradient :
- contoh penerapan di algoritma Support Vector Machine (SVM)
- Menentukan garis pembatas antar 2 kelas
Kalkulus: turunan & optimisasi
- Turunan (differensial) merupakan pengembangan dari gradient. Turunan dari sebuah fungsi didefinisikan sebagai berikut :
         df/dx:=lim h→0 (f(x+h)-f(x)) /h
- Turunan sering digunakan untuk mencari nilai maksimum atau minimum
- Titik maksimum/minimum ditemukan dengan menemukan x yang memenuhi f(x) =0
- Nilai maksimum/minimum digunakan untuk memperkecil error atau memaksimalkan jarak 2 kelas yang berbeda.

Konsep dasar :
- Ruang sample (S): himpunan dari semua kemungkinan dari suatu percobaan 
- Peluang suatu kejadian A adalah banyaknya himpunan A dibagi banyaknya himpunanS.
        P(A) = n(A)/n(S)
Peluang Bersyarat : 
- 2 kejadian A dan B dikatakan saling lepas jika AnB = 0, misalnya kejadian munculnya angka ganjil dan genap saat melempar 1 dadu. Pada kejadian ini P(An B) =0.
- Jika kejadian A tidak mempengaruhi kejadian B (dan sebaliknya) maka A dan B dikatakan saling bebas. Peluang munculnya kejadian A dan B = P(ANB)=P(A) X P(B) Misalnya kejadian terjadinya kecelakaan dan hujan
- Peluang kejadian A atau B = P(AUB)=P(A) + P(B) - P(ANB) 
- Peluang terjadi suatu kejadian H bila diketahui bahwa kejadian X telah terjadi disebut peluang bersyarat yang dikenal dalam teorema bayes
         P(H|X) = P(H n X) / P(X) =P(X|H) P(H) / P(X)

Formula Matematis ANN : 
Neural Network satu lapis disebut perceptron. Perceptron memberikan output tunggal. 
- x0,x1,..x(n) mewakili berbagai masukan (variabe bebas). Masing-masing input dikalikan dengan bobok koneksi atau sinapsis. 
- Bobot dipresentasikan sebagai w0,w1,…w(n). Bobot menunjukkan kekuatan mode tertentu. 
- b adalah nilai bias yang memungkinkan untuk menggeser fungsi aktivitas ke atas atau ke bawah. 
Perceptron 
The Perceptron : Forward Propagation 
Input -> Weights -> Sum -> Non- linearity -> output
   Activation Function membantu menyelesaikan masalah yang bersifat non-trivial dalam suatu neural network dengan mengambil sebuah nilai dan melakukan operasi matematika. Fungsi matematika pada ANN adalah ReLu dan Sigmoid. Pentingnya activation function yaitu dapat memperkenalkan non linieritas ke dalam jaringan. 
   Loss Function digunakan untuk mengukur seberapa bagus performa dari neural network dalam melakukan prediksi terhadap target. Loss / cost function merupakan fungsi yang menggambarkan kerugian yang terkait dengan semua kemungkinan yang dihasilkan oleh model.
   Optomizer adalah fungsi pengoptimalan yang dapat digunakan untuk memperbarui weight network secara iteratif berdasarkan data training. Contohnya yaitu gradient descent, SGD, momentum, dan ADAM optimizer .
   Back propagation adalah proses memperbaharui bobot dari network untuk mereduksi error dalam hasil prediksi. 
   Gradient descent adalah proses menyesuaikan parameter model untuk turun melalui Loss function.

Confusion Matrix :
- Realita (yes) , Prediksi (yes) : True Positive (TP).
- Realita (no), Prediksi (yes) : False Positive (FP).
- Realita (yes), Prediksi (no) : False Negative (FN). 
- Realita (no), Prediksi (no) : True Negative (TN). 
   Akurasi adalah persentase prediksi yang benar dari semua kemungkinan
Akurasi = (TP + TN) / (TP + FP + TN + FN) 
              = Prediksi benar / Semua case
   Presisi adalah persentase dari case yang diprediksi positif (TP + FP) yang ternyata positif (TP). 
Presisi = (TP) / (TP + FP)
            = True Positive / Total prediksi positif 
   Recall mengukur pecahan kasus positif (TP + FN) yang diidentifikasi dengan benar (TP). 
Recall = (TP) / (TP + FN)
            = True Positive / Total aktual positif

Pembagian Statistik
- Deskriptif : Mendeskripsikan dan menganalisis sampel data tanpa melakukan proses penarikan kesimpulan
- Inferensial : Menganalisa sampel untuk mengestimasi dan membuat kesimpulan mengenai populasi
   Ukuran pemusatan adalah metode yang umum digunakan untuk mengidentifikasi pusat kumpulan data. 
Ukuran tendensi sentral :
- Mean atau rata-rata : jumlah semua nilai dalam kumpulan data, dibagi dengan jumlah total nilai dalam kumpulan data.
- Median adalah nilai tengah dari kumpulan data jika jumlah nilai adalah bilangan ganjil; rata-rata dari dua nilai tengah, jika jumlah nilai adalah bilangan genap.
- Mode : Nilai yang paling sering muncul dalam kumpulan data, yaitu nilai sampel acak yang terjadi dengan frekuensi terbesar
   Ukuran Penyebaran : Digunakan untuk mengetahui bagaimana data tersebar di sekitar mean
Ukuran variabilitas : 
- Range : Jarak antara nilai terkecil dan nilai terbesar dalam sebuah dataset.
- Kuartil dan Inter-quartile range
- Varians adalah ukuran seberapa menyebar set data
- Standar deviasi : Jarak rata-rata semua nilai dari rata-rata keseluruhan.

Confusion Matrix :
- Realita (yes) , Prediksi (yes) : True Positive (TP).
- Realita (no), Prediksi (yes) : False Positive (FP).
- Realita (yes), Prediksi (no) : False Negative (FN). 
- Realita (no), Prediksi (no) : True Negative (TN). 
   Akurasi adalah persentase prediksi yang benar dari semua kemungkinan
Akurasi = (TP + TN) / (TP + FP + TN + FN) 
              = Prediksi benar / Semua case
   Presisi adalah persentase dari case yang diprediksi positif (TP + FP) yang ternyata positif (TP). 
Presisi = (TP) / (TP + FP)
            = True Positive / Total prediksi positif 
   Recall mengukur pecahan kasus positif (TP + FN) yang diidentifikasi dengan benar (TP). 
Recall = (TP) / (TP + FN)
            = True Positive / Total aktual positif

   Ukuran pemusatan adalah metode yang umum digunakan untuk mengidentifikasi pusat kumpulan data. 
Ukuran tendensi sentral :
- Mean atau rata-rata : jumlah semua nilai dalam kumpulan data, dibagi dengan jumlah total nilai dalam kumpulan data.
- Median adalah nilai tengah dari kumpulan data jika jumlah nilai adalah bilangan ganjil; rata-rata dari dua nilai tengah, jika jumlah nilai adalah bilangan genap.
- Mode : Nilai yang paling sering muncul dalam kumpulan data, yaitu nilai sampel acak yang terjadi dengan frekuensi terbesar
   Ukuran Penyebaran : Digunakan untuk mengetahui bagaimana data tersebar di sekitar mean
Ukuran variabilitas : 
- Range : Jarak antara nilai terkecil dan nilai terbesar dalam sebuah dataset.
- Kuartil dan Inter-quartile range
- Varians adalah ukuran seberapa menyebar set data
- Standar deviasi : Jarak rata-rata semua nilai dari rata-rata keseluruhan.

- Skala Nominal : Data berupa kategorik, namun tidak berurutan. Pada data ini, statistik yang cocok adalah frekuensi dan modus 
        Contoh : laki-laki dan perempuan 
- Skala Ordinal : Data bersifat kategorik, namun memiliki makna urutan. Pada data ini, statistik yang cocok adalah frekuensi, modus, dan median
        Contoh : tingkat pendidikan 
2.) Numerical 
- Skala Interval : Data numerik tanpa 0 mutlak. Statistik yang cocok untuk analisa datanya adalah frekuensi, mean, median, modus, varians
        Contoh : data suhu
- Skala Rasio : Data numerik dengan 0 mutlak.  Statistik yang cocok untuk analisa datanya adalah frekuensi, mean, median, modus, varians
         Contoh : data kecepatan

   Class adalah kumpulan objek. Class berisi cetak biru atau prototipe dimana objek akan dibuat. Entitas logis yang berisi beberapa atribut dan metode.
- Classes dibuat dengan keyword class
- Attributes adalah variable yang dimiliki sebuah class
- Attributes selalu bersifat public dan bisa diakses menggunakan operator dot(.)
Syntax :

OOPs terminology : Object, instance, method, constructor method, class variable, instance variable
Python class dan objects :
- keyword class digunakan untuk membuat sebuah class
- Membuat objek dari class yang ada
OOPs terminology : self keyword
1. Self mewakili instance dari kelas
2. Method dari sebuah class harus memiliki tambahan parameter dalam definisinya. Tidak harus memberikan nilai untuk parameter ini.
3. Dapat mengakses atribut dan metode class dengan Python.
4. Self mirip dengan pointer di C++ dan reference di Java.
Methods inside the class :
- methods yang didefinisikan di dalam class dengan keyword Self sebagai input 
- Mengakses atribut class untuk dibagikan ke berbagai instance dari class yang sama
