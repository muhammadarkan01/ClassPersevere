Kemudian mempelajari tentang  Jupyter Notebook. Jupyter Notebook adalah salah satu IDE untuk penelitian dan pengembangan, di bidang data science, yang berbasis web. IDE ini bersifat open sources. Umumnya, program yang ditulis menggunakan Jupyter Notebook akan disimpan dalam file dengan format .ipynb. Tetapi, kita juga menyimpan program yang kita tulis dalam format .py apabila dibutuhkan.
Selanjutnya mempelajari tentang program dan programming. Program adalah urutan instruksi (code) yang menentukan bagaimana cara komputer melakukan komputasi.
Komputasi di sini dapat berupa:
• Sesuatu yang berhubungan dengan matematika • Komputasi simbolis
Programming adalah aktivitas menulis intruksi (code) untuk dijalankan oleh komputer. Contoh instruksi seperti:
• Mencetak “Hello World”
• Mengambil data dari internet (scraping) 
• Membaca isi file
Code dapat ditulis menggunakan bahasa pemrograman (programming language.
Programming language adalah alat/tools yang digunakan untuk menulis code. Programming language tersusun atas serangkaian simbol yang berfungsi menerjemahkan pikiran kita menjadi instruksi yang dapat dipahami komputer.
Dahulu, programming sulit dilakukan, karena menggunakan bahasa pemrograman tingkat rendah (low-level programming language), seperti bahasa assembly.
Code lebih dekat ke kode mesin (biner: 0 dan 1) ketika ditulis menggunakan low-level language.
 Pada saat ini, umumnya code ditulis menggunakan bahasa pemrograman tingkat tinggi (high-level programming language), seperti Python, Java, C, C++, dll.
Code lebih dekat ke bahasa manusia (English) ketika ditulis menggunakan high-level language.
Programming language dibangun di atas abstraksi. Abstraksi adalah konsep untuk mengelola kompleksitas. Contoh:
Ketika menulis program Python, kita tidak perlu tahu bagaimana cara Python berinteraksi dengan operating system (OS), bagaimana cara OS berinteraksi dengan hardware, dan bagaimana cara hardware mengeksekusi binary.
Pada dasarnya, kita melakukan programming di atas beberapa layer abstraksi.
Semakin tinggi level programming language, maka: 
• semakin jauh dari kode mesin;
• semakin banyak abstraksi di bawahnya; dan
• semakin lambat berjalan.

Code lebih dekat ke kode mesin (biner: 0 dan 1) ketika ditulis menggunakan low-level language. Pada saat ini, umumnya code ditulis menggunakan bahasa pemrograman tingkat tinggi (high-level programming language), seperti Python, Java, C, C++, dll.
Code lebih dekat ke bahasa manusia (English) ketika ditulis menggunakan high-level language. Programming language dibangun di atas abstraksi. Abstraksi adalah konsep untuk mengelola kompleksitas. Contoh: Ketika menulis program Python, kita tidak perlu tahu bagaimana cara Python berinteraksi dengan operating system (OS), bagaimana cara OS berinteraksi dengan hardware, dan bagaimana cara hardware mengeksekusi binary. Pada dasarnya, kita melakukan programming di atas beberapa layer abstraksi. Semakin tinggi level programming language, maka: 
• semakin jauh dari kode mesin;
• semakin banyak abstraksi di bawahnya; dan
• semakin lambat berjalan.

   Video pertama yang saya tonton adalah penjelasan tentang apa itu phyton. Phyton adalah salah satu jenis bahasa pemrograman yang saat ini populer untuk digunakan. Alasan phyton menjadi sangat populer yaitu mudah digunakan dan di aplikasikan, memiliki banyak library/pustaka yang menjadikan bahasa ini jadi sangat efisien dan mudah untuk membuat Pemograman tingkat menengah hingga lanjutan, dan bersifat open source. Bahasa Pemograman ini didukung oleh program berorientasi objek dan juga bahasa Pemograman tingkat tinggi. Yang dimaksud dengan bahasa Pemograman tingkat tinggi adalah bahasa komputer yang memakai instruksi berasal dari unsur kata-kata manusia seperti if, else, dll dalam mengerjakan proses pengcodingannya. Guido van Rossum adalah seseorang yang telah membuat dan mengembangkan bahasa pemrograman phyton pada tahun 1990 an. Kemudian versi pengembangan phyton yang lebih lanjut dilakukan pada tahun 1995 agar phyton lebih kompatibel untuk digunakan. Selanjutnya pada tahun 2000 phyton terus dilakukan pembaharuan hingga mencapai versi 3 atau phyton 3 sampai saat ini.
Konon inspirasi guido van rossum memberikan nama phyton berasal dari sebuah acara tv favorit nya yang bernama Mothy Phyton Flying Circus. 
Phyton banyak diaplikasikan di industri dan perusahaan besar teknologi, seperti Google, NASA, Yahoo,dll. Banyak perusahaan di masa depan yang akan membutuhkan keterampilan pemograman khususnya phyton. Code editor berbasiskan phyton notebook yang akan digunakan yaitu Jupyter dan Google Colab.
