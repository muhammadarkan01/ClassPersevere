- Mampu menangani data multidimensi dan multivariasi.
- Aplikasi/implementasi yang luas.

- Akuisisi data

Neural Network --> Harus punya hidden layer biar jadi deep learning. 
*Backpropagation adalar proses memperbaharui bobot dari network untuk mereduksi error dalam hasil prediksi.
*Gradient Descent adalah proses menyesuaikan parameter model untuk turun melalui loss function. 
*Loss function digunakan untuk mengukur seberapa bagus performa dari neural network dalam melakukan prediksi terhadap target.
*Optimizer adalar fungsi pengoptimalan yang dapat digunakan untuk memperbaharui beban secara iteratif berdasarkan data training.
Kenapa angka pada gambar 0 – 255? Baik hitam putih atar berwarna? Karena 1 warna = 8 bits —> 2^8 = 256.
Kalau hitam putih mika dikalikan ada 256 warna. Sedangkan yang berwarna jika dikalikan sekitar 16 juta warna.
- SVM —> dipakai untuk klasifikasi gambar, komputasi lebih ringan.
