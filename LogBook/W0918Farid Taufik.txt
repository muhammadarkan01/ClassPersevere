Thu, 21 Apr

Hari ini saya mengikuti sesi pembelajaran di Orbit AI Mastery
saya mempelajari:

- Modul Domain AI - Reinforcement Learning
- Bagian Review and Forum Group Discussion.
- Konsep RL: Teori Dasar => RL belajar (learn) dari interaksi agent dengan environment menggunakan proses eksploitasi/eksplorasi dan sistem reward, untuk memperkuat (reinforce) action positif.
- Konsep RL: Interaksi Agent => Tujuan dari agent adalah memaksimalkan reward kumulatif, disebut return.
- Konsep RL: Policy => Policy adalah algoritme/serangkaian aturan yang mendeskripsikan bagaimana agent mengambil keputusan-keputusan.
- Konsep RL: Value Function => Value function berguna untuk mengevaluasi posisi agent, baik atau buruk, pada state yang diketahui.
- Konsep RL: Model => Karena agent umumnya tidak dapat mengamati environment seluruhnya, agent harus melakukan action untuk mendapatkan informasi tentang environment (model-free).
- MDP (Markov Decision Process) => Markov Reward Process yang memiliki keputusan-keputusan action.
- Persamaan Bellman: Formulasi, Perhitungan.
- Monte Carlo (Konsep MC) => tidak mengambil pengetahuan lengkap dari environment (model free).
- Elemen Algoritme MC => Goal, Return, Value Function.
- MC Control => Policy Evaluation & Policy Improvement.
- Temporal Difference (TD) => TDL : Mengestimasi reward pada setiap Langkah (step).
- Deep Q Learning => Deep Q Learning akan menggantikan Q-Table menggunakan sebuah Neural Network yang biasa disebut dengan Deep Q Network atau DQN. 
- Deep Q Network => sebuah NN yang menerima states yg diberikan oleh environment sebagai input, lalu DQN akan menghasilkan output estimasi Q Values pada setiap actions yang dapat diambil pada state tersebut.
