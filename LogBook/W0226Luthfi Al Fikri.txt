- Python Function
Fungsi memberikan modularitas yang lebih baik untuk aplikasi Anda dan tingkat penggunaan kode yang tinggi.'
- Build-in function
ada 2 Cara dari Build function in Python : Standard dan Operasi Matematika.
ada 2 Type Build Function : Type Conversation dan Utility based function.
- User Defined Function
Fungsi ini bekerja dari metode utama untuk menata kode dan dapat digunakan kembali dalam python.
Blok pernyataan yang menerima beberapa argumen, melakukan beberapa fungsi, dan menyediakan output.
Keyword def digunakan untuk membuat fungsi yang ditentukan pengguna dengan python.
Hasil fungsi ini dinyatakan dengan return().
- String Manipulation
String baris tunggal  dapat dibuat menggunakan tanda kutip tunggal dan ganda, String multiline dapat dibuat menggunakan tanda kutip tiga.
Python menyediakan banyak seperangkat operator, fungsi, dan metode untuk bekerja dengan string.
- File Handling
Python mempunyai fungi untuk menangani file seperti menulis, membaca, merubah dan menghapus file
Fungsi awal untuk dapat melakukan operasi file adalah open() yang mempunyai 2 parameter yaitu filename dan mode.
filename adalah nama file yang ingin diproses sedangkan mode adalah metode operasi yang ingin dilakukan. 
Ada 3 jenis metode operasi dasar di Python yaitu :
* r =	read - Untuk membaca file
* a =	append - Untuk menambah data
* w =	write - Untuk menulis data
