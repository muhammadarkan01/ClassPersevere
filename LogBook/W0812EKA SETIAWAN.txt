Tue, 12 Apr

Image senoising with Autoencoder. Autoencoder adalah algoritme pembelajaran mesin tanpa pengawasan yang menerapkan propagasi balik, menyetel nilai target agar sama dengan input. Components Autoencoder ada Encoder, Code, Decoder. Encoder yaitu bagian jaringan memampatkan input menjadi latent space representation. Code yaitu bagian dari jaringan yang mewakili input terkompresi yang diumpankan ke Decoder. Decoder yaitu bertujuan untuk merekonstruksi input dan Laten space representation. Properties Autoencoders ada Data-specific, Lossy, Unsupervised. Data-specific yaitu Autoencoder hanya mampu mengompresi data secara bermakna serupa dengan apa yang telah dilatih. Lossy yaitu output dari Autoencoder tidak akan sama persis dengan input, akan menjadi representasi yang dekat tetapi terdegradasi. Unsupervised yaitu untuk melatih Autoencoder tidak perlu melakukan sesuatu yang mewah, cukup melemparkan data input mentah. Hyperparameters Autoencoder ada Code size, Number of layers, Loss function, Number of node per layers. Architecture Autoencoder ada Input Imag (Encoder), Latent space Representation (Bottleneck), Reconstructed Image (Decoder).
