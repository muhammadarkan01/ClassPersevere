Pada sesi 1 belajar mengenai sejarah dari bahasa pemrograman python, cara penggunaannya python pada jupyter notebook serta memahami konsep dari AI. Wed, 2 Mar

Materi pembelajaran pada sesi pertama yaitu mengenai python operator. Operator pada bahasa pemrograman python terdiri dari, operator aritmatika, operator assignment, operator comparison (perbandingan), operator logical, operator identity, operator membership dan operator bitwise. Operator digunakan untuk melakukan operasi pada variabel atau nilai. Dalam sesi pertama juga dijelaskan mekanisme pengumpulan tugas dengan menggunakan git dan github. Kemudian pada sesi kedua, materi yang disampaikan yaitu perkondisian seperti if else serta perulangan seperti for, while.

Fri, 4 Mar

Hari ini mempelajari mengenai cara membuat sebuah fungsu pada python. Fungsi pada python dapat dinyatakan dengan def, pernyataan def harus diikuti dengan nama fungsi dan daftar parameter formal dalam kurung. Pada materi ini juga dilakukan membuat fungsi python seperti fungsi luas segita, persegi dan sebagainya. Selain mempelajari fungsi pada python, pada kegiatan hari ini juga dibahas mengenai fungsi bawaan atau built-in function yang sudah tersedia pada python.
