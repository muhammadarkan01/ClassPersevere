mengakhiri Hari  mempelajari ANN (artificial neural network).ANN merupakan pemrosesan informasi yang terinsipirasi dari Sistem syaraf biologi manusia,ANN belajar melalui contoh data yang diberikan.

ANN sendiri terdapat 3 layer yaitu Input Layer (masukkan),Hidden Layer dan Output layer (keluaran)

Arsitektur neural network terbagi menjadi 3 kelas yaitu :
- single layer feed forward neural  networks (terdiri dari 3 Input node dan  4 output node)
- multi layer feed forward neural networks (terdiri dari 3 input node 4 hidden node dan 2 output node)
-reccurent neural networks

kemudian yang terakhir di materi ANN ada Fungsi aktifasi,Terdapat 2 tipe dari fungsi aktifkasi yang ada yaitu Linear activations functions dan non linear Activation functions.perbedaan nya terlihat cukup jelas jika kita pindahkan ke Grafik Linear. Linear functions akan terlihat linear di grafik garis, sedangkan non linear akan melengkung dengan banyak titik (dot ) yang ada disekitarnya
