Kegiatan hari ini dimulai dengan meet virtual mempelajari introduction AI dilanjut dengan materi Python Introduction. Jam 10-12 dilanjutkan dengan belajar mandiri di w3schools yaitu Python Turtorial mulai dari Python Home – Python Dictionaries dan juga mengerjakan exercise yang disediakan di w3schools untuk mengasah materi yang sudah dibaca. Setelah ISOMAN dilanjutkan meet virtual melanjutkan pembahasan Python Introduction dan dari pembelajaran tersebut dapat diringkas :

-	Struktur data adalah cara untuk menyimpan data dalam bahasa pemrograman.
-	Python memiliki 4 tipe data primitf yang tidak dapat diubah integer, float, string, dan boolean.
-	Python memilki 4 tipe data non primitive. List, tuple, dictionary, dan set.
-	Tuple tidak dapat diubah sedangkan list, dictionary, dan set dapat diubah.
-	Kita dapat mengetikkan tipe data primitive dari satu ke yang lain.
