Wed, 30 Mar

Hari ini pembelajaran melalui zoom dengan membahas:
Data Science:
• Penjelasan mengenai recommender system yang digunakan berbagai aplikasi, seperti gojek, Instagram, dll. Recomemder system bekerja berdasarkan karakteristik user dan/atau item. Item yang memiliki karakter sama maka akan direkomendasikan.
• menjelaskan mengenai teknik recommender system seperti market basket yang diterapkan untuk teknik marketing produk.
NLP:
• Penjelasan mengenai text classification sebuah proses pemberian tag atau kategori ke teks menurut isinya.
• penjelasan machine learning approach merupakan proses klasifikasi teks dengan melatih mesin untuk mempelajari pola pada kumpulan teks. Mampu menentukan polaritas sebuah teks/ sentiment baru learning. Memiliki kekurangan pada dimensi fitur yang tinggi, fitur noise, dan pelabelan data.
• Penjelasan Lexicon-based approach merupakan proses klasifikasi teks dengan menggunakan kamus kata yang sudah didefinisikan sebelumnya. Memiliki kelebihan tanpa pelabelan data dan proses komputasi lebih cepat. Terdapat kekurangannya seperti harus mendaftarkan semua kata dalam kamus kata. Jika belu terdaftar, sangat mempengaruhi perfoma klasifikasi.
• Hands on implementasi text classification

Fri, 1 Apr

Hari ini pembelajarn melalui zoom dengan membahas:
Kelas Reinforcement Learning:
• Penjelasan mengenai TDL yang memiliki idea dasar berupa kombinasi DP dan MC. TDL sendiri diaplikasikan pada model free dan melakukan update value state per state. TDL berisfat bootstapping dengan menggunakan metode trial dan error dalam explorasi lingkungan dan cepat dalam mencapai konvergensi. TD control menggunakan SARSA dan !-Learning, SARSA bekerja berdasarkan perubahan state-action (S.A). SARSA merupakan algoritma on policy dan pengambilan keputusan dari state-action SARSA berdasarkan epsilon-greedy policy.
Kelas Technical pembahasan tentang Web Scraping:
• Penjelasan web scraping, scraping sendiri suatu kegiatan mengambil informasi dari situs web atau layer computer dan memasukannya ke dalam spreadsheet di computer. Jadi web scraping merupakan praktik pengumpulan data melalui cara apapun selai dari menggunakan program untuk berinteraksi denga API.
• Pembahasan mengenai kegunaan web scraping seperti pada situs web E-commerce untuk mengumpulkan data yang terkait dengan produk tertentu dari berbagai situs web e-commerce untuk melakukan perbandingan. Pada marketing & sales untuk mendapatkan data seperti email, nomor telepon, dll. Untuk kampanye penjualan dan pemasaran serta iklan. Pada search engine optimization (SEO) untuk memberi tahu bisnis bagaiman peringkat mereka untuk kata kunci pencarian yang penting bagi pemilik bisnis. Dan pada Data untuk proyek ML sebagai pengambilan dataset untuk proyek pembelajaran mesin banyak bergantuk pada web scraping.
• Penjelasan mengenai perbedaan Web Scraping Vs Web Crawling, salah satu perbebdaannya seperti pada web scraping mendapatkan informasi sepesifik, sedangkan pada web s=crawling mendapatkan informasi umum.
• Penjelasan Beautiful soup merupakan library python yang dirancang untuk proyek web scraping.
• Penjelasan Urlib3 merupakan library python lain yang dapat digunakan untuk mengambil data dari URL mirip dengan request library.
• Penjelasan Scrapy sebagai kerangka kerja web crawling open-source yang cepat dan dibuat dengan python, digunakan untuk mengekstrak data dari halaman web dengan bantuan penyeleksi berdasarkan XPath.
• Penjelasan Selenium suatu kumpulan aplikasi open source untuk pengujian otomatis aplikasi web di berbagai browser dan platform.
