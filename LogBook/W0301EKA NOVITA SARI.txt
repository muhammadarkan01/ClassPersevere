Mon, 7 Mar

Pada sesi pertama materi hari ini yaitu memahami AI atau kecerdasan buatan dan evolusinya, memahami hubungan antara AI, machine learning dan deep learning, menghubungkan bidang lain seperti data science, computer vision dan NLP dengan AI, memahami konsep dibalik machine learning, dan memahami proses standar dari machine learning. Pada materi dijelaskan bahwa data adalah kumpulan fakta-fakta yang ada disekitar kita. Data merupakan bentuk jamak dari kata datum, kumpulan fakta dan statistik dalam bentuk mentah atau tidak teroganisir, serta kumpulan fakta dan statistik yang digunakan untuk referensi dan analisis. Selain itu juga ada digital data yang berupa informasi yang diproses atau disimpan oleh komputer dalam bentuk dokumen teks, gambar, audio, dan lain-lain. Dilanjutkan dengan membahas artificial intelligence data science.
Kemudian pada sesi kedua pembahasan nya kelanjutan dari sesi pertama tadi yaitu tentang artificial intelligence computer vision, natural language processing, dan reinforcement learning. Menjelaskan tentang game AI Moral Machine juga. Lalu membahas tentang apa itu machine learning, kenapa machine learning, applications of machine learning, programming languages, machine learning cloud platforms, what to learn in machine learning, dan yang terakhir ada quiz dan kesimpulan. Setelah sesi kedua selesai saya lanjutkan dengan melihat video pembelajaran yang ada pada orbit guru.

Tue, 8 Mar

Hari ini mempelajari tentang matematika dasar untuk kecerdasan buatan yaitu peluang, aljabar linier, dan kalkulus. Di dalam aljabar linier saya mempelajari tentang vektor dan skalar, menghitung jarak dan sudut antar vektor, operasi matriks, determinant & invers, penerepan invers, nilai & vektor eigen. Pada materi kalkulus saya mempelajari tentang fungsi & gradient, turunan & optimisasi, dan contoh penggunaan gradient. Sedangkan pada peluang saya mempelajari tentang konsep dasar, dan peluang bersyarat. Dilanjutkan dengan tebak-tebakan matematika dan materi lanjutan operasi matematika & NumPy yaitu contoh penggunaan, indexing & silicing (pemisahan), operasi matematika seperti penjumlahan/pengurangan array & dot product vektor, perkalian matriks, mean, median, nilai maksimum, sorting, transpose, determinan, invers, eigen value & vector, generate matriks yang entrinya 0 semua ukuran pxq, generate matriks identitas ukuran pxp, terakhir dilanjutkan dengan pembuatan program di jupyter notebook. 

Wed, 9 Mar

Hari ini saya belajar tentang semgemtasi ML yang terdiri dari supervised learning, unsupervised learning, traditional programming vs machine learning approach. Terdapat 7 langkah dalam ML yaitu gathering data, preparing data, choosing a model, training model, evaluation, tuning hyperparameter, dan prediction.  Kemudian ada metrics evaluation yang terdiri dari regression, classification, unsupervised, dan others. Saya juga mempelajari kelebihan dan kekurangan ML, ciri-ciri unsupervised learning, selain itu juga ada review concept, machine learning vs deep learning, neural network, deep learning, perkembangan deep learning, ML vs DL, dan lain-lain. Kemudian dilanjutkan dengan materi artificial neural network, mengetahui tentang kelebihan neural network, mengetahui cara kerja ANN, mempelajari formula matematis ANN, activation function, back propagation, loss function, dan optimizer.

Thu, 10 Mar

Materi yang dipelajari hari ini yaitu tentang AI project cycle yang terdiri dari system thinking, system map, algoritma, dan flow. Yang dibahas pertama tentang AI project cycle yaitu problem scoping atau masalah yang harus dipecahkan. Kemudian data acquisition untuk mengetahui data apa yang dibutuhkan. Data exploration ,  modelling yang digunakan untuk merepresentasikan masalah ke dalam bentuk yang lebih sederhana, dan yang terakhir ada evaluation, dan deployment. Selanjutnya materi kedua yang dibahas pada hari ini yaitu tentang data analytics with python-basic statistics. Yang pertama ada jenis data, data terbagi menjadi tiga, yaitu data structured, semi-structured, dan unstructured. Kemudian ada statistik deskriptif yang membahas ukuran pemusatan yang digunakan untuk mengidentifikasi pusat kumpulan data menggunakan mean, median, mode. Dan ukuran penyebaran yang digunakan untuk mengetahui bagaimana data tersebar di sekitar mean, yang biasanya digunakan dalam data kuantitatif dengan langkah-langkah penyebaran yaitu range, kuartil dan inter-quartile range, varians, dan standar deviasi. Selanjutnya saya juga mempelajari tentang skala pengukuran data yang terdiri dari skala nominal yang berupa data kategori tanpa urutan, ordinal yang berupa data kategori dengan urutan, interval yang berupa data numerik tanpa 0 mutlak, dan rasio yang berupa data numerik dengan 0 mutlak. Dan yang terakhir tentang status multivariat yang terdiri dari kovarians dan korelasi.

Fri, 11 Mar

Materi hari ini mempelajari tentang OOPs and Common Packages, dimulai dengan membahas tentang object oriented programming, class atau kumpulan objek, terminologi, python class dan objects (dipraktikkan pada jupyter notebook),  self keyword, methods inside the class, inheritance (dipraktikkan pada jupyter notebook), dan pemberitahuan perihal tugas ke-5. Selanjutnya pada materi kedua membahas tentang modules yang terdiri atas python custom modules, dan python built in modules. Python modules merupakan file dengan ekstensi .py yang berisi kode python yang dapat di import ke dalam program python lain. Keuntungan dari modules yaitu reausability, simplicity, dan scoping. Kemudian mempraktikkan kode program python custom modules di dalam jupyter notebook. Sedangkan python built-in modules merupakan python yang memiliki banyak fungsi bawaan, selain itu banyak fungsi yang tersedia sebagai bagian dari perpustakaan yang dibundel dengan distribusi python, mayoritas modul python ditulis dalam bahasa C dan terintegrasi dengan python shell. Kemudian dilanjutkan dengan mempraktikkan kode program modules os, time, dan math. Selain itu pada materi hari ini juga membahas tentang section recap maths, basic & advanced statistics. Pada materi ini dijelaskan tentang scalar, vektor, matrix, tensor, serta visualisasi vektor, ruang vektor, let's exploring other dimensions (menonton video youtube bersama perihal exploring other dimensions), machine learning, dan percepatan sebagai vektor. Selanjutnya membahas tentang recap AI, ML, dan DL, terdapat perbandingan antara traditional programming vs machine learning (menonton video youtube bersama yang berjudul intro to machine learning), dilanjutkan membahas tentang machine learn from data, traditional programming vs machine learning approach, machine learning vs deep learning, deep learning: training and testing, how neural network learn: backpropagation,  regression vs classification, digit classification MNIST dataset, dan the fashion MNIST dataset.
