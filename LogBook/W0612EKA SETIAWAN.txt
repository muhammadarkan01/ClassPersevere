Tue, 29 Mar

Pagi 
Mereview materi Image Filtering. Proses Konvolusi pada image filtering. Contoh Konvolusi. Filter Kernel Low pass filter (membuang frekuensi tinggi pada citra), High pass filter (membuang frekuensi rendah pada citra digunakan untuk deteksi tepi), Band stop filter (meloloskan frekuensi tertentu digunakan untuk efek sharpness). Melakukan Lave Coding. Deep Learning di Image. Convolution Neural Networks yaitu kategori Neural Network yang efektif di berbagai bidang seperti pengenalan dan klasifikasi gambar. Dapat mengidentifikasi wajah, objek, dan rambu lalu lintas, robot vision, self driving car, dan smart grid. Sejarah CNN diciptakan oleh Yann LeCunn, model CNN pertama disebut LeNet pada tahun 1988. Cars recognition, Self driving car, dll. Topologi CNN memiliki pebedaan dari ANN standard dimana neuron disusun menjadi 3 dimensi. Architecture CNN. Convolution Layer bertujuan untuk mengekstarak fitur dari citra masukan, mempertahankan hubungan special antara input dengan mempelajari fitur input. Stride yaitu parameter yang menentukan berapa jumlah pergeseran filter. Padding / zero padding yaitu parameter menentukan jumlah pixel yang berisi nilai 0 yang akan ditambahkan di setiap sisi dari input. Layer ReLU / Rectified Linear Unit merupakan tipe dari fungsi aktivasi. Layer Pooling yaitu mengurangi jumlah parameter ketika input terlalu besar. Jenis pooling : Max pooling, Average pooling, Sum pooling.

Siang 
Mengerjakan Pre-Test. Mereview materi minggu lalu mengenai MDP concept, DP Algorithm (mengaplikasikan algoritme DP dibutuhkan pengetahuan tentang model environment) DP -> Model-based algorithm. Solusi mengimplementasikan RL tanpa memodelkan environmentnya secara lengkap disebut Model free algorithm -> Monte Carlo. Konsep Monte Carlo (MC) contoh Dadu simetris (Outcome : 1, 2, 3, 4, 5, 6. Probability : 1/6, 1/6, 1/6, 1/6, 1/6, 1/6). Dadu tidak simetris (Outcome : 1, 2, 3, 4, 5, 6. Probability : n1/N, n2/N, n3/N, n4/N, n5/N, n6/N). MC tidak mengambil pengetahua lengkap dari environment, belajar dari experience, episode per episode. Elemen Algoritme MC ada Goal, Return, Value Function. First-visit MC yaitu dihitung saat pertama kali mengunjungi state s pertama dalam setiap episode. Every-visit MC yaitu dihitung setiap mengunjungi state s dalam setiap episode. MC Estimation (karena model tidak tersedia, perlu untuk mengestimasi action juga, selain hanya mengestimasi statenya, pada MC action diperlukan dalam menentukan policy. Policy Evaluation Problem yaitu expected retrun mulai dari state, melakukan action, dan mengikuti policy). Forzen Lake Problem (satu episode berhenti ketika agent mencapai tujuan/ goal, agent mendapat reward 1 ketika mencapai tujuan dan 0 untuk kondisi yang lain). Melakukan Hands On MCForzenLake dan Mengerjakan Post Test.
